# Laboratoire 7.1.3 - configurations
## Informations

Ce sont les configurations de base (Part 1 - Step 2) Ã  utiliser pour le laboratoire.   

**Attention :** N'oubliez pas d'ajuster les interfaces selon vos appareils.  

## Configuration de base

#### Router R1
```running-config
hostname R1
no ip domain lookup
line con 0
 logging sync
 exec-time 0 0
 exit
banner motd # This is R1, OSPFv2 Route Summarization and Filtering Lab #
! Ajuster pour GNS3
interface g0/0
 ip add 172.16.0.2 255.255.255.252
 no shut
 exit
! Ajuster pour GNS3
interface GigabitEthernet0/1
 ip address 10.10.0.1 255.255.255.252
 no shut
 exit
router ospf 123
 router-id 1.1.1.1
 auto-cost reference-bandwidth 1000
 network 10.10.0.0 0.0.0.3 area 1
 network 172.16.0.0 0.0.0.3 area 0
 exit
```
