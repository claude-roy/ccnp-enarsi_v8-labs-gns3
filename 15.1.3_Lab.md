## Configurations de base

#### R1 Configuration  
```running-config
hostname R1
no ip domain lookup
line con 0
 logging sync
 exec-time 0 0
 exit
banner motd # This is R1, Path Control Using PBR #
interface G0/0
 description Connection to R2
 ip add 172.16.0.2 255.255.255.252
 no shut
 exit
interface G0/2
 description Serial Connection to R3
 ip add 172.16.2.1 255.255.255.252
 no shut
 exit
interface G0/1
 description Connection to D1
 ip add 10.10.0.1 255.255.255.252
 no shut
end
``` 

#### R2 Configuration

#### D1 Configuration  
```running-config
hostname D1
no ip domain lookup
line con 0
exec-timeout 0 0
logging synchronous
exit
banner motd # This is D1, Path Control Using PBR #
interface e0/1
 no switchport
 description Connects to R1
 ip address 10.10.0.2 255.255.255.252
 no shut
 exit
interface Loopback 1
 description Interface simulates network
 ip ospf network point-to-point
 ip address 10.10.1.1 255.255.255.0
 exit
interface Loopback 2
 description Interface simulates network
 ip ospf network point-to-point
 ip address 10.10.2.1 255.255.255.0
end
```
