## Configurations de base

#### R1 Configuration  
```running-config
hostname R1
no ip domain lookup
line con 0
 logging sync
 exec-time 0 0
 exit
banner motd # This is R1, Controlling Routing Updates #
interface g0/0
 description Connection to R2
 ip add 172.16.0.2 255.255.255.252
 no shut
 exit
interface Lo1
 ip address 172.16.1.1 255.255.255.0
exit
interface Lo12
 ip address 172.16.12.1 255.255.255.192
exit
interface Lo13
 ip address 172.16.13.1 255.255.255.224
exit
interface Lo14
 ip address 172.16.14.1 255.255.255.240
exit
interface Lo15
 ip address 172.16.15.1 255.255.255.248
end
``` 

#### R2 Configuration  
```running-config
hostname R2
no ip domain lookup
line con 0
 logging sync
 exec-time 0 0
exit
banner motd # This is R2, Controlling Routing Updates #
interface g0/0
 description Connection to R1
 ip add 172.16.0.1 255.255.255.252
 no shut
exit
interface GigabitEthernet0/1
 description Connection to R3
 ip address 192.168.0.1 255.255.255.252
 no shut
exit
int lo10
 ip add 10.10.10.1 255.255.255.0
 ip ospf network point-to-point 
end
```
